Angular Pre-requisites Installation

Pre-requisites

1. Node.js

2. Angular CLI

3. Visual Studio Code Editor

Step 1 : Download the Node.js software from the below link

Click here to download

Step 2 : Install the Node.js software

Node.js is an open-source, cross-platform JavaScript run-time environment that executes JavaScript code outside of a browser.

Step 3 : Verify the Node.js and NPM installation

Go to command prompt and type the below commands to verify the installation.

C:\Windows\System32>node -v

v10.13.0

C:\Windows\System32>npm -v

6.4.1

NPM (Node Package Manager) is a package manager for the JavaScript programming language. It is the default package manager for the JavaScript runtime environment Node.js

Note: The version number may change for newer Node.js
