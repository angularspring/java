Adding bootstrap to Angular 7

We have seen angular setup and project creation in previous tutorial. The next step is to add botstrap in angular project and use it in efficient way. Hope, you have already installed and created a new project. So, we are skipping those parts to make this tutorial simple.

Step 1 : Install Bootstrap and FontAwesome

We will use npm to install the dependencies:

npm install bootstrap font-awesome

Now that the dependencies are installed we need to load these dependencies to our project.

Step 2 : Load CSS dependencies to our project

To load Bootstrap CSS dependencies to our project, we have to mention the dependencies name in angular.json file.

We have to add bootstrap.min.css and font-awesome.min.css in to 'styles' section in angular.json.

angular.json

"styles": [
"src/styles.css",
"node_modules/bootstrap/dist/css/bootstrap.min.css",
"node_modules/font-awesome/css/font-awesome.min.css"
],

Step 3 : Load JS dependencies to our project

To use full feature of bootstrap we need to add JS dependencies also.

To load Bootstrap JS dependencies to our project, we have to mention the dependencies name in angular.json file.

We have to add bootstrap.bundle.min.js in to 'scripts' section in angular.json. bootstrap.bundle.min.js is a combination of bootstrap.js and popper.js.

Bootstrap JS is depending on Jquery. So, we need to add jquery dependency first.

Using npm we can install jquery to our project.

npm install jquery

Now, we have to add both jquery and bootstrap.bundle.min.js files in angular.json.

angular.json

"scripts": [
"node_modules/jquery/dist/jquery.min.js",
"node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"
],

Step 4 : Verify the installation of bootstrap

After adding bootstrap to angular.json file, stop the server and start again with ng serve command.

Note : Whenever you are doing changes in angular.json file, please stop and start server to see the changes.

Before applying bootstrap, home page will look like this.

After started the server, the home page will look like this. You can able to indentify the font related changes happened.
