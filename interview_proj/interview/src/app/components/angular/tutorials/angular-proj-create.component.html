Angular 7 - Project Creation

Welcome to Angular world, we have seen angular setup and editor setup in previous tutorial. The next step is create a new angular project and explore the new dimension of web coding.

Step 1 : Create a new workspace

Create a new workspace (folder) in 'D:' or 'E:' drive.

Ex: D:\Angular

Step 2 : Navigate to workspace folder

Open the command prompt in administrator mode and go to the workspace location.

C:\Windows\System32>D:

D:\>cd training

D:\training>

Step 3 : Create a new Angular project using Angular CLI

create a new angular project by executing the below command in cmd prompt.

Syntax : ng new project-name

D:\training>ng new angular-training

This will create a new project in the name of 'angular-training'

Step 4 : Choosing Routing and Style options

After executing the above command, it will ask two option.

The first option is about angular routing. If you want to use Routing in your project, then Type 'y' or 'yes'. All projects needed routing, so best practice to go with routing option.

The second option is about CSS file types. Select the first one ‘CSS’ and enter to proceed. If you want to use mixin format, then go with either 'SCSS' or 'SASS' options.

? Would you like to add Angular routing? Yes

? Which stylesheet format would you like to use? CSS

After the choice chooses, it will complete the project creation. It will look like below screenshot

Step 5 : Start the server to run Angular project

Go into the project folder to start the server.

D:\training>cd angular-training

Now start the server using the below command.

D:\training\angular-training>ng serve –o

After this command execution, it will open the website in browser in default port 4200. You can directly open the site by entering the below url in browser.

http://localhost:4200

The webpage will be look like below

If you want to run the project in different port number, then use the below command.

ng serve --port=5600

This command will start the server in port number 5600.

Step 6 : Open the angular code in Visual studio code.

Open the visual studio code and navigate to File menu.

File -> Open Folder -> Select the folder name of angular-training from ‘D:’ drive.

After select the project folder, it will open the code in Visual Studi Code Editor.

Your first angular project has been created and hosted locally.

Happy Coding!!!
