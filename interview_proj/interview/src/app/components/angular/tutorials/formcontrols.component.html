Adding Form Controls and Validation

Step 1 : Importing Form Module

Import FormsModule into root module of our project. Always root module will be app.module.ts

app.module.ts:

....
import {{'{'}} FormsModule } from '@angular/forms';
....
@NgModule({{'{'}}
  imports:[
    ....
    FormsModule
  ],
  .....
})
export class AppModule {{'{'}}}

FormsModule will enable the ngModel directive to our project to achive the two way binding. FormsModule will provide all forms related directives and validations.
